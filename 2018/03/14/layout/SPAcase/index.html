SPA搭建购物车案例

在线预览地址：https://xiamuqing.github.io/shoppingCar/

SPA

single page web application
只有一张web页面，加载单个HTML 页面并在用户与应用程序交互时动态更新该页面的Web应用程序。
其核心：通过哈希的改变从而加载不同的页面。

功能

  • 商品展示功能
  • 添加商品至购物车功能
  • 修改购物车中商品数量
  • 删除购物车中商品

用到的技术

  • SPA
  • seajs
  • artTemplate

其他

无数据库，数据写在JS文件中
