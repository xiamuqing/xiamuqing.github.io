TypeScript实现heap

堆就是用数组实现的二叉树
堆常用方法：

  • 构建优先队列
  • 支持堆排序
  • 快速找出集合中的最小值（或最大值）

分为：最大堆和最小堆，两者差异在于节点的排序方式，在最大堆中，父节点的值比每一个子节点的值都要大。在最小堆中，父节点的值比每一个子节点的值都要小。

目的：将最大（最小）的节点放在最前面，从而进行快速的插入、删除等操作。在二叉树中搜索会很快，但是在堆中搜索会很慢。

注意：堆的根节点中存放的是最大或者最小元素，但是其他节点的排序顺序是未知的。例如，在一个最大堆中，最大的那一个元素总是位于 index 0 的位置，但是最小的元素则未必是最后一个元素。唯一能够保证的是最小的元素是一个叶节点，但是不确定是哪一个。

parent(i) = Math.floor((i - 1)/2)
left(i)   = 2i + 1
right(i)  = 2i + 2

class CreateMaxHeap {
    type: string
    arr: number[]

    /**
     * @description 创建一个大顶堆，不改变原数组
     */
    constructor(arr: number[], type: string = 'MAX') {
        let newArr = []
        if (Array.isArray(arr)) {
            newArr = [...arr]
            if (newArr.length) {
                newArr = this.init(newArr)
            }
            this.arr = newArr
            this.type = type
        }
    }

    /**
     * 
     * @param arr 需要调整的数组
     * @description 返回大顶堆，且改大顶堆，左节点比右节点大
     */
    init (arr: number[]): number[] {
        // 堆的高度，排序遍历的次数，堆的层级为 height+1
        const heapHeight = Math.floor(arr.length / 2)
        // 将null插入数组最前，这样方便使用index计算
        arr.unshift(null)
        for (let i = heapHeight; i > 0; i--) {
            this.update(arr, i, arr.length - 1)
        }
        arr.shift()

        return arr
    }

    /**
     * 从index开始检查并保持大顶堆
     * @param arr  待检查数组
     * @param i    检查的起始下标
     * @param s 堆大小
     * @description 调整大顶堆
     */
    update (arr: number[], i: number, s?: number): void {
        let size = s ? s : arr.length
        let left = 2 * i
        let right = left + 1

        // 左右孩子中较大的那一个
        let maxlr = -1

        //无左右孩子节点
        if (left > size && right > size) {
            return
        }
        //只有左孩子节点
        if (left <= size && right > size) {
            maxlr = left
        }
        //只有右孩子节点
        if (right <= size && left > size) {
            maxlr = right
        }
        //同时有左右孩子节点
        if (left <= size && right <= size) {
            maxlr = arr[left] < arr[right] ? right : left
        }

        if (arr[i] < arr[maxlr]) {
            this.swap(arr, i, maxlr)
            this.update(arr, maxlr, size)
        }
    }

    swap (arr: number[], i: number, j: number): void {
        let temp = arr[i]
        arr[i] = arr[j]
        arr[j] = temp
    }

    /**
    * @description 返回堆排序好的新数组，不影响原数组
    */
    sort (): number[] {
        const arr = this.init([...this.arr])
        arr[0] !== null && arr.unshift(null)
        for (let j = arr.length - 1; j > 1; j--) {
            //将堆顶与堆底元素交换，分离出最大的元素
            this.swap(arr, 1, j)
            //重新调整大顶堆剩余未分离的堆
            this.update(arr, 1, j - 1)
        }
        arr.shift()
        return arr
    }

    /**
     * @param i 需要移动的元素节点索引
     * @description 将当前节点元素层层向上交换位置
     */
    shiftUp (arr: number[], i: number): void {
        const parent = Math.floor((i - 1) / 2)
        if (arr[i] < arr[parent] || parent < 0) {
            return
        } else {
            this.swap(arr, i, parent)
            this.shiftUp(arr, parent)
        }
    }

    /**
     * @param el 插入的元素
     * @description 返回插入好的大顶堆
     */
    insert (el: number): number[] {
        this.arr.push(el)
        const i = this.arr.length - 1
        this.shiftUp(this.arr, i)
        return this.arr
    }

    /**
     * @param i 被删除元素的坐标
     * @description 删除指定坐标的元素
     */
    removeByIndex (i: number): number {
        const j = this.arr.length - 1
        // 将被删的元数和数组最后一个元素交换位置
        this.swap(this.arr, i, j)
        const el = this.arr.pop()
        this.update(this.arr, i)
        return el
    }
    /**
     * @param el 被删除元素
     * @description 删除找到的第一个指定元素
     */
    remove (el: number): number[] {
        const i = this.arr.indexOf(el)
        if (el !== -1) {
            this.removeByIndex(i)
        }
        return this.arr
    }

    /**
     * @param i 被替换元素的坐标
     * @param el 替换的内容
     * @description 替换指定位置的元素
     */
    replace (i: number, el: number): void {
        let orgin = this.arr[i]
        this.arr.splice(i, 1, el)
        if (el > orgin) {
            // 新来的元素比原来的大，往上走
            this.shiftUp(this.arr, i)
        } else {
            // 新来的元素比原来的小，往下走
            this.update(this.arr, i)
        }
    }

    /**
     * @description 弹出最大值
     */
    pop (): number {
        const arr = this.arr;
        arr.unshift(null);
        this.swap(arr, 1, arr.length - 1);
        const top = arr.pop();
        let lastParentIndex = Math.floor((arr.length - 1) / 2);
        for (let i = lastParentIndex; i > 0; i--) {
            this.update(arr, i, arr.length - 1);
        }
        arr.shift();
        return top;
    }

}

const maxHeap = new createMaxHeap(arr);

maxHeap.insert(8);

maxHeap.remove(8);
console.log(maxHeap.arr); // [10, 6, 4, 5, 2, 1, 3]
maxHeap.replace(2, 8);
console.log(maxHeap.arr); //[10, 6, 8, 5, 2, 1, 3]
console.log("sort", maxHeap.sort()); //[1, 2, 3, 5, 6, 8, 10]
console.log(maxHeap.arr); // [10, 6, 8, 5, 2, 1, 3]

有一堆石头，每块石头的重量都是正整数。

每一回合，从中选出两块 最重的 石头，然后将它们一起粉碎。假设石头的重量分别为 x 和 y，且 x <= y。那么粉碎的可能结果如下：

如果 x == y，那么两块石头都会被完全粉碎；
如果 x != y，那么重量为 x 的石头将会完全粉碎，而重量为 y 的石头新重量为 y-x。
最后，最多只会剩下一块石头。返回此石头的重量。如果没有石头剩下，就返回 0。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/last-stone-weight\

function lastStoneWeight(stones) {
    const sts = new CreateMaxHeap(stones);
    // console.log(sts.arr);
    while (sts.arr.length > 1) {
        const one = sts.pop();
        const two = sts.pop();
        const diff = one - two;
        if (diff > 0) {
        sts.insert(diff);
        // console.log("insert", sts.arr);
        }
    }

    return sts.arr[0] || 0;
    }
